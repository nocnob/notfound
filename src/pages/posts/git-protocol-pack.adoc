== 准备

启动 Git Daemon 服务，允许通过 git 协议推、拉代码：

[source,bash]
----
git daemon \
    --verbose \
    --export-all \
    --base-path=$PWD \
    --reuseaddr \
    --enable=receive-pack  \
    --listen=127.0.0.1 
----
* `--verbose` 输出详细信息
* `--export-all` 目录下的所有仓库都可以拉取
* `--base-path` 仓库根目录
* `--reuseaddr` 允许服务重启时不等待旧的连接超时
* `--enable` 允许的服务，这里允许仓库推送
* `--listen` 监听的地址和端口号
